Provides an intention that converts Kotlin lambda braces into parentheses and vice versa to simplify editing and refactoring of inline lambda expressions.

This intention allows you to quickly switch between { } and ( ) when working with Kotlin lambdas. It detects the bracket under or near the caret—including when the caret is placed immediately before or after the bracket—and applies a consistent transformation. Both directions are supported:

  • { ... } → ( ... )
  • ( ... ) → { ... }

This is especially useful when adjusting lambda syntax, converting between trailing-lambda style and explicit-argument style, or manipulating DSL-like lambda blocks.

Example:


// Before:
listOf(1, 2).filter { it % 2 == 0 }

// After applying the intention:
listOf(1, 2).filter ( it % 2 == 0 )

// And the reverse direction also works.

Text below this line appears only in the intention settings inside the IDE.

You can adjust intention visibility or enable/disable intentions through Settings → Editor → Intentions.
